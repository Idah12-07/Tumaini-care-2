# DHIS2 / KHIS Kenya — Tumaini Indicator Reference

## How to find indicator UIDs
1. Log in at https://hiskenya.org
2. Go to Maintenance > Data Elements > Indicators
3. Search for the indicator name
4. Copy the UID (10-character alphanumeric code)

## Key indicators needed for Tumaini
| Indicator | DHIS2 name to search | Why Tumaini needs it |
|---|---|---|
| ANC 1st visit | ANC 1st visit | Baseline pregnancy volume by county |
| Miscarriage count | Abortion / Miscarriage | EPL burden by facility |
| Maternal death | Maternal death | Mortality calibration |
| CHW visits | Community health visit | CHW capacity baseline |
| Postpartum visit | PNC visit | Existing follow-up benchmark |
